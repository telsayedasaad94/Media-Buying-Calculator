# Media Buying Calculator

حاسبة وسائط متقدمة وشاملة لحساب المقاييس الإعلانية وتحليل الحملات الإعلامية.

Advanced and comprehensive media buying calculator for calculating advertising metrics and analyzing media campaigns.

## ✨ المميزات الرئيسية | Main Features

### 1. **حاسبة أساسية** (Basic Calculator)
- حساب عدد الانطباعات (Impressions)
- حساب عدد النقرات (Clicks)
- حساب التحويلات (Conversions)
- حساب الإيرادات الكلية
- حساب العائد على الاستثمار (ROI)
- معادلات أساسية:
  - **CPC** = الميزانية الكلية / النقرات
  - **CPA** = الميزانية الكلية / التحويلات
  - **ROAS** = الإيرادات / الميزانية

### 2. **تحليل متقدم** (Advanced Analysis)
- اختيار منصة الإعلان (Facebook, Google, TikTok, Instagram, LinkedIn)
- حساب مدة الحملة والميزانية اليومية
- حساب التكرار (Frequency)
- حساب معدل الانخراط (Engagement Rate)
- حساب نقطة التعادل (Breakeven Point)
- رسم بياني لاتجاهات الأداء

### 3. **تخطيط الميزانية** (Budget Planning)
- توزيع الميزانية على عدة قنوات (Facebook, Google, TikTok, Other)
- توزيع يومي للميزانية
- جداول تفصيلية للتخطيط

### 4. **مقاييس الأداء** (Performance Metrics)
- بيانات الأداء الأسبوعية
- بيانات الأداء اليومية
- مقارنة الحملات
- عرض حالة كل حملة (إيجابية/سلبية)

## 📊 المتغيرات المستخدمة | Variables

### متغيرات الحاسبة الأساسية:
- **Total Budget**: الميزانية الكلية (بالجنيه المصري)
- **CPM**: تكلفة الألف انطباع
- **CTR**: معدل النقر (%)
- **Conversion Rate**: معدل التحويل (%)
- **AOV**: متوسط قيمة الطلب

### الصيغ المستخدمة | Formulas:
```
Impressions = (Budget / CPM) × 1000
Clicks = Impressions × (CTR / 100)
Conversions = Clicks × (Conversion Rate / 100)
Revenue = Conversions × AOV
ROI = ((Revenue - Budget) / Budget) × 100
CPC = Budget / Clicks
CPA = Budget / Conversions
ROAS = Revenue / Budget
```
